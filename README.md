# ToDo_Application

### Features :

- Add todo item in specific category

- Create new category

- Edit and delete todo items

- Todo item count in each category

- Email notification when new todo_item is created

- Reminder email before item expires.



### Technology stack :

- Python

- Django

- Sqlite

- CSS3

- Bootstrap
