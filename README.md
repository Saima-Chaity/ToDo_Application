# ToDo_Application

This is a simple todo application in Python and Django.

### Features :

- Add todo item in specific category

- Create new category

- Edit and delete todo items

- Todo item count in each category


### Technology stack :

- Python

- Django

- Sqlite

- CSS3

- Bootstrap
